  • HTML documents are described by HTML tags

css

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

from W3Schools.com

Examples

							
h1 {
    color: red;
    font-size: 40px;
}

p {
    color: blue;
    font-size: 12px;
}

img {
    margin: 10px;
}

Examples

							
<p class="hero">Brief description of Eddie Jaoude ...</p>
<p>Full description of Eddie Jaoude ...</p>
							
						
							
p {
    color: blue;
    font-size: 12px;
}

p.hero {
    color: red;
    font-size: 20px;
}

Create the file style.css

Add the following inside the <head> section

							
								<link rel="stylesheet" href="styles.css">

Full page example

							
<!DOCTYPE html>
<html>
	<head>
		<title>Title of the document</title>
		<link rel="stylesheet" href="styles.css">
	</head>

	<!-- My visible page -->
	<body>
		<h1>Eddie Jaoude</h1>
		<p class="hero">Brief description of Eddie Jaoude ...</p>
		<p>Full description of Eddie Jaoude ...</p>
	</body>
</html>

Bootstrap

Bootstrap is the most popular HTML, CSS, and JS framework in the world for building responsive, mobile-first projects on the web.

What does it do

  • Responsive layout
  • Lots of helper components
  • Alerts
  • Forms
  • Buttons
  • Navigation
  • Modal
  • ...

Setup

Documentation http://v4-alpha.getbootstrap.com
